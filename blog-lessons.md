# Blog lessons

One sentence per shipped post. Keeps the learning compounding.

- 2026-05-14 (eagain-not-a-linker-error): a Field Note can stand without a singular debug session if the underlying mechanism is well-documented in primary sources; cite the man pages and let the post be an explainer, not a war story.
